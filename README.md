# atgame

## Usage
Clone this repository
```bash
git clone https://github.com/anixne/cprojects
```

### Windows
```
bin\atgame.exe
```
### Linux
to run the executable file you will need wine.
```bash
wine bin/atgame.exe
```


### Gameplay
W - up
S - down
A - left
D - right.

Enjoy.
